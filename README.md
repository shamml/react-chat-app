_Десктопный чат_

---
_Запуск проекта_

Для запуска проекта вам необходимо набрать команду в терминале: npm i, после уже npm start

---


_Основной функционал приложения_
- Отправка сообщений
- Удаление сообщений
- Поиск сообщений
- Выбор чата
- Поиск контактов
- Профиль активного чата

---
_Технологии проекта_
- React  
- Redux 
- Redux-Thunk 
- React-Router 
- Prettier 
- PropTypes 
- dayJs 
- CssTransition
- Git
